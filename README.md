# 🛒 Blinkit Sales Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive Power BI dashboard built to analyze and visualize sales performance for Blinkit (India’s last-minute delivery app).

The dashboard provides actionable insights into sales trends, outlet performance, product categories, and customer preferences, enabling data-driven decision-making.

---

## 🎯 Problem Statement

Retail and quick-commerce businesses generate massive amounts of data daily, but:

Data is often scattered across multiple sources
No real-time visibility into performance
Difficult to track KPIs across outlet types, sizes, and locations
Lack of insights into product category performance

👉 The goal of this project is to centralize, analyze, and visualize sales data in a single dashboard for better business understanding.

---

## 📊 Key KPIs

The dashboard focuses on the following critical metrics:

| KPI | Value |
|-----|------|
| 💰 Total Sales | $1.20M |
| 📦 No. of Items | 8,523 |
| ⭐ Avg Rating | 4.0 |
| 📈 Avg Sales | $141 |

-----

## 📈 Dashboard Features
**1. Sales Performance Analysis**
- Year-wise outlet establishment trend
- Identification of peak sales years
- Growth pattern visualization

**2. Product Category Insights**
- Sales breakdown by item type
- Top-performing categories:
- Fruits & Vegetables
- Snack Foods
- Household items

**3. Fat Content Analysis**
- Comparison between Low Fat vs Regular items
- Contribution to total sales
- Outlet-wise fat category performance
  
**4. Outlet Analysis**
- 🏪 By Size: High, Medium, Small
- 📍 By Location Tier: Tier 1, Tier 2, Tier 3
- 🏬 By Type: Supermarket Type1, Type2, Type3, Grocery Stores

**5. Interactive Filters**
- Outlet Size
- Outlet Location
- Item Type

---

## 🧠 Approach
**Step 1: Data Understanding**
- Reviewed dataset structure (sales, ratings, outlet details, item categories)
  
**Step 2: Data Cleaning & Transformation**
- Removed inconsistencies
- Standardized categories
- Created calculated columns and measures

**Step 3: Data Modeling**
- Built relationships between tables
- Optimized for performance

**Step 4: DAX Calculations**

Used DAX to create:
- Total Sales
- Average Sales
- KPI cards
- Category-wise aggregations

**Step 5: Dashboard Design**
- Focused on clarity + usability
- Used consistent color theme (Blinkit-inspired)
- Balanced charts for storytelling

---

## 🔍 Key Insights
- Tier 3 outlets generated the highest sales contribution
-Medium-sized outlets contributed the most revenue (~42%)
- Regular fat products dominated (~65% of total sales)
- Fruits & Vegetables and Snack Foods are top revenue drivers
- Supermarket Type1 outlets perform best overall
